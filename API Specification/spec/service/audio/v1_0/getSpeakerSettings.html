audio - v1.0 None

getSpeakerSettings Â¶

This API provides current settings and supported settings related to speaker configuration items.
Syntax
http://<Base URL>/audio
ws://<Base URL>/audio
Authentication Level
none
  • Request
  • Response
  • Notification
  • See also
params' Elements
An object composed by following pair(s).
name type multiplicity default description
target string ? ""
Target name. Default value is "". It means settings of all targets. Client can get all speaker settings information by setting empty string to "target". Client can get specific speaker setting value by explicitly setting target name to "target".
  • "" - Settings of all targets.
  • "inCeilingSpeakerMode" - Set whether or not to use in-ceiling speaker mode with the current input. In a setup where the front and center speakers are installed in the ceiling, lowering the audio output position to the screen height allows you to enjoy a more natural experience. (more...)
  • "tvPosition" - Setting the sound according to the TV position. (more...)
  • "speakerSelection" - Select the front speaker system to operate. (more...)
  • "frontLLevel" - Set level of Front Left speaker. (more...)
  • "frontRLevel" - Set level of Front Right speaker. (more...)
  • "centerLevel" - Set level of Center speaker. (more...)
  • "surroundLevel" - Set level of Surround speaker. (more...)
  • "surroundLLevel" - Set level of Surround Left speaker. (more...)
  • "surroundRLevel" - Set level of Surround Right speaker. (more...)
  • "surroundcBackLevel" - Set level of Surround Back speaker. (more...)
  • "surroundBackLLevel" - Set level of Surround Back Left speaker. (more...)
  • "surroundBackRLevel" - Set level of Surround Back Right speaker. (more...)
  • "frontHighLLevel" - Set level of Front High Left speaker. (more...)
  • "frontHighRLevel" - Set level of Front High Right speaker. (more...)
  • "subwooferLevel" - Set level of Subwoofer speaker. Note that the range and step values vary depending on device. (more...)
  • "subwooferFreq" - Adjust the cut off frequency of the Wireless Subwoofer. All frequencies below the cut off frequency are output to the Wireless Subwoofer instead of the TV speakers. (more...)
  • "subwooferPhase" - Sets the phase polarity of subwoofer. (more...)
  • "subwooferPower" - Set the power control method of the Wireless Subwoofer. (more...)
JSON Example
{
    "id": 67,
    "method": "getSpeakerSettings",
    "params": [{"target": "frontLLevel"}],
    "version": "1.0"
}

deviceUIInfo string ? ""
This parameter provides information how devices display UI information. This format is product dependent and actual format should be explained in Device Specific Info page .
title string ? ""
Title of settings for display purpose. "" means no specified title.
titleTextID string ? ""
The identifier, which is unique in product, assigned by server to identify title. "" means no specified title text ID.
type string ? ""
Type of value to identify actual value type of target.
  • "enumTarget" - Enumeration type for specifying finite candidates. (ex) "high", "mid", "low"
  • "booleanTarget" - Boolean type only two candidates available. (ex) "off, "on"/"false", "true"
  • "integerTarget" - Integer type. (ex) "1", "-10"
  • "doubleNumberTarget" - Number type including double defined in Scalar Web API. (ex) "1.5", "-10.0"
  • "stringTarget" - String type. (ex) "hello"
  • "" - no type information.
isAvailable boolean ? true
This target is currently available or not.
candidate (object-array) ? null
Candidates of specified settings. Only If there is no candidate, null is set.
  title string ? ""
Title of candidate for UI display purpose. "" means no specified title
  titleTextID string ? ""
The unique identifier, which is unique in product, assigned by server to identify title. "" means no specified title text ID.
  value string ? ""
Value of candidate. "" can be set only when settings value is integer and the candidate can be represented by "max", "min", and "step".
  isAvailable boolean ? true
This candidate is currently available or not.
  max double ? -1
Max value of specified settings. Only If value of target is not indicated by numerical number, -1 is set.
  min double ? -1
Min value of specified settings. Only If value of target is not indicated by numerical number, -1 is set.
  step double ? -1
Step value of specified settings. Only If value of target is not indicated by numerical number, -1 is set.
