recording - v1.0 Private

getScheduleList Â¶

This API provides the list of items scheduled both for recording and reminder.
Syntax
http://<Base URL>/recording
ws://<Base URL>/recording
Authentication Level
private
  • Request
  • Response
  • Notification
  • See also
params' Elements
An object composed by following pair(s).
name type multiplicity default description
stIdx integer ? 0
Start index to get the list.
cnt integer ? 100
Count of the maximum number of items that can be listed, starting from "stIdx".
JSON Example
{
    "id": 30,
    "method": "getScheduleList",
    "params": [{
        "cnt": 20,
        "stIdx": 0
    }],
    "version": "1.0"
}
result's Elements
An array of objects composed by following pairs.
name type multiplicity default description
id string 1  
ID to identify this scheduled item.
type string 1  
Type of schedule
uri string 1  
URI to identify channel.
  • (ex) "tv:isdbt?trip=11.22.33"
title string 1  
Name of scheduled item. TV's program name is usually set, but the naming rule depends on each device.
channelName string 1  
Name of the broadcasting channel.
startDateTime string 1  
Scheduled date and time to start. The format is ISO-8601 as described below. (YYYY-MM-DDTHH:MM:SS.sTZD)
durationSec integer 1  
Duration in seconds.
repeatType string 1  
Repeat type

Supported repeatType values can be retrieved by using getSupportedRepeatType.
overlapStatus string 1  
Overlap Status
recordingStatus string 1  
Recording Status
quality string ? "DR"
Recording mode for video quality.
  • "DR" - Direct recording
JSON Example
{
    "id": 30,
    "result": [[
        {
            "id": "1",
            "channelName": "NHKE",
            "title": "Let'sgolunch",
            "startDateTime": "2012-10-10T11:45:00+0000",
            "durationSec": 1800,
            "repeatType": "w15",
            "type": "recording",
            "uri": "tv:isdbt?trip=11.22.44",
            "overlapStatus": "notOverlapped",
            "recordingStatus": "notStarted"
        },
        {
            "id": "2",
            "channelName": "BBCi",
            "title": "SaturdayNightFever!",
            "startDateTime": "2012-10-13T22:00:00+0000",
            "durationSec": 7200,
            "repeatType": "w6",
            "type": "recording",
            "uri": "tv:isdbs?trip=33.22.44",
            "overlapStatus": "fullyOverlapped",
            "recordingStatus": "recording"
        }
    ]]
}
Error Code
No additional error code is defined. Refer to error code for common errors.
