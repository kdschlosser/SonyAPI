recording - v1.0 Generic

deleteSchedule Â¶

This API provides function to delete existing recording/reminder item.

If an error code is returned, target item fails to be deleted and still remains in the schedule list.
Syntax
http://<Base URL>/recording
ws://<Base URL>/recording
Authentication Level
generic
  • Request
  • Response
  • Notification
  • See also
params' Elements
An object composed by following pair(s).
name type multiplicity default description
id string 1  
ID to identify this scheduled item. ID can be retrieved by GetScheduleList.
type string 1  
Type of schedule
uri string 1  
URI to identify channel.
  • (ex) "tv:isdbt?trip=11.22.33"
title string 1  
Name of scheduled item. Set "title" value in the result of getScheduleList.
startDateTime string 1  
Scheduled date and time to start. The format is ISO-8601 as described below. (YYYY-MM-DDTHH:MM:SS.sTZD)
durationSec integer ? 0
Duration in seconds. This parameter can be optional only when type is "reminder". In such case, default value is 0 which means no specific end time for reminder.
JSON Example
{
    "id": 44,
    "method": "deleteSchedule",
    "params": [{
        "id": "3",
        "title": "Showtime",
        "startDateTime": "2012-10-13T22:00:00+0000",
        "durationSec": 3600,
        "type": "recording",
        "uri": "tv:isdbt?trip=11.22.33"
    }],
    "version": "1.0"
}
result's Elements
Not necessary.
JSON Example
{
    "id": 44,
    "result": []
}
Error Code
Following error codes are important for this API. Refer to error code for other errors.
code Explanation
41200
Scheduled item specified in the request does not exist.
41201
Middle of recording.
41232
program will start in less than 30 seconds.
