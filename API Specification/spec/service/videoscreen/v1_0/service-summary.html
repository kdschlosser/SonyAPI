Service videoScreen Â¶

This service handles APIs that are related to video screen functions.
    API List
    API name Auth Level Description
    getAudioSourceScreen (v1.0)
    None
    This API provides which video screen has sound output.
    getBannerMode (v1.0)
    None
    This API provides function to get current banner mode and the list of banner mode values which are able to be set.
    getMultiScreenMode (v1.0)
    None
    This API provides current screen mode of device.
    getMultiScreenMode (v1.1)
    None
    This API provides current screen mode of device.
    getPipSubScreenPosition (v1.0)
    None
    This API provides position of sub screen in PIP mode.
    getSceneSetting (v1.0)
    None
    This API provides function to get current scene setting value and the list of scene setting values which are able to be set.
    requestToNotifyScreenState (v1.0)
    None
    This API provides a function to notify screen state change of device.
    requestToNotifyScreenState (v1.1)
    None
    This API provides a function to notify screen state change of device.
    setAudioSourceScreen (v1.0)
    Generic
    This API provides a function to change sound output among multiple video screens.
    setBannerMode (v1.0)
    Generic
    This API provides a function to change the current banner mode value.
    setMultiScreenMode (v1.0)
    Generic
    This API provides a function to change the screen mode of device.
    setPapScreenSize (v1.0)
    Generic
    This API provides a function to change the screen size in PAP mode.
    setPipSubScreenPosition (v1.0)
    Generic
    This API provides a function to change the sub screen position in PIP mode.
    setSceneSetting (v1.0)
    Generic
    This API provides a function to change the current scene setting value.
    Common parameters in videoScreen service
    Multi Screen Mode
    Multi Screen Mode refers to the mode in which multiple video screens are located on the video output of device in our terminology. The explanation of each mode and the simple example are described in the following figures and tables.



    "single"
    single screen mode
    "PIP"
    Picture In Picture mode
    "PAP"
    Picture And Picture mode
    "widgetMode"
    Picture and Widget application mode
    "internetPhoneMode"
    Internet Phone application mode
    Screen
    To identify one of the possible video screens, following values are defined.
    "main"
    main screen
    "sub"
    sub screen
