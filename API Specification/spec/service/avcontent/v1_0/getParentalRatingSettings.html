avContent - v1.0 Generic

getParentalRatingSettings Â¶

This API provides a function to get the current parental rating settings of the device. It is possible to set multiple parental rating settings in the device.
Note
Based on country, different rating setting is received from device. In US, ratingTypeSony and custom values (ratingCustomTypeMpaa and ratingCustomTypeTV would be set only when ratingTypeSony is custom.) In Canada, ratingTypeSony and custom values (ratingCustomTypeCaEnglish and ratingCustomTypeCaFrench and ratingCustomTypeMpaa would be set only when ratingTypeSony is custom.) In other (EU or Japan etc.), age can only be received.
Syntax
http://<Base URL>/avContent
ws://<Base URL>/avContent
Authentication Level
generic
  • Request
  • Response
  • Notification
  • See also
params' Elements
Not necessary.
JSON Example
{
    "id": 1,
    "method": "getParentalRatingSettings",
    "params": [],
    "version": "1.0"
}
result's Elements
An object composed by following pair(s).
name type multiplicity default description
ratingTypeAge integer ? -1
Age of restriction. This is used in Japan or Europe (other than ATSC). Default value is -1. (It means that the device does not have this setting value.)
ratingTypeSony string ? ""
Rating setting for sony.
  • "" - The device does not have this setting value.
  • "off" - Rating setting is off.
  • "child" - All children
  • "youth" - Parental Guidance
  • "youngAdult" - Strongly cautioned by parents
  • "custom" - Select to set more restrictive ratings.
ratingCountry string ? ""
Country of rating setting. Default value is "". (It means that the device does not have this setting value.)
  • "" - The device does not have this setting value.
  • "us" - United States
  • "ca" - Canada
ratingCustomTypeTV string-array ? null
Rating setting of TV. Selected types are listed. If no types are selected, array is empty [ ]. Default value is null, means the device does not have this setting value.
  • "y" - All children
  • "y7" - Directed to older children
  • "g" - General audience
  • "pg" - Parental Guidance suggested
  • "14" - Parents Strongly cautioned
  • "ma" - Mature Audience only
ratingCustomTypeMpaa string ? ""
MPAA setting for US (MPAA: Motion Picture Association of America) ("off" / "g" / "pg" / "pg-13" / "r" / "nc-17" / "x" / "nr"). Default value is "". (It means that the device does not have this setting value.)
  • "" - The device does not have this setting value.
  • "off" - Rating setting is off.
  • "g" - General audience
  • "pg" - Parental Guidance
  • "pg-13" - Parents strongly cautioned
  • "r" - Restricted
  • "nc-17" - No one 17 and under admitted
  • "x" - No one 17 and under admitted
  • "nr" - Not Rated
ratingCustomTypeCaEnglish string ? ""
English rating setting for Canada ("off" / "e" / "c" / "c8" / "g" / "pg" / "14" / "18"). Default value is "".
  • "" - The device does not have this setting value.
  • "off" - Rating setting is off.
  • "e" - Exempt.
  • "c" - Children
  • "c8" - Children 8 years and older
  • "g" - General programming
  • "pg" - Parental Guidance
  • "14" - Viewers 14 and older
  • "18" - Adult programming
ratingCustomTypeCaFrench string ? ""
French rating setting for Canada ("off" / "e" / "g" / "8" / "13" / "16" / "18"). Default value is "". (It means that the device does not have this setting value.)
  • "" - The device does not have this setting value.
  • "off" - Rating setting is off.
  • "e" - Exempt.
  • "g" - General.
  • "8" - Not recommended for younger children.
  • "13" - Not recommended for children under age 13.
  • "16" - Not recommended for ages under 16.
  • "18" - This program is restricted to adults.
unratedLock boolean ? false
If it is true, the content without rating information is locked. If it is false, the content without rating information is not locked.
JSON Example
In the case sony rating is used in US region 
{
    "id": 1,
    "result": [{
        "ratingTypeSony": "youth",
        "ratingCustomTypeCaEnglish": "off",
        "ratingCountry": "us",
        "ratingCustomTypeTV": [],
        "unratedLock": true,
        "ratingCustomTypeCaFrench": "off",
        "ratingCustomTypeMpaa": "off"
    }]
}
In the case custom rating is used in Canada region 
{
    "id": 1,
    "result": [{
        "ratingTypeSony": "custom",
        "ratingCustomTypeCaEnglish": "pg",
        "ratingCountry": "ca",
        "ratingCustomTypeTV": ["y7"],
        "unratedLock": false,
        "ratingCustomTypeCaFrench": "off",
        "ratingCustomTypeMpaa": "off"
    }]
}
For others 
{
    "id": 1,
    "result": [{"ratingTypeAge": 13}]
}

No additional error code is defined. Refer to error code for common errors.
