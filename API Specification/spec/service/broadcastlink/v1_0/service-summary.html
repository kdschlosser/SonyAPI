Service broadcastLink Â¶

This service handles APIs that satisfy hybridcast (defined as part of broadcast specification in arib region and in isdbg region in the near future.) The APIs defined in this service are related between server device and companion application (assumed running on another device).
    API List
    API name Auth Level Description
    getBroadcastLinkServerInfo (v1.0)
    None
    This API provides a function to get information of broadcastLink server.
    requestToNotifyEvent (v1.0)
    Generic
    This API provides a function to start monitoring of event notified from broadcastLink service.
    sendMessage (v1.0)
    Generic
    This API provides a function to send a message from companion device to a broadcastlink server.
    setConnectionMode (v1.0)
    Generic
    This API provides a function to declare client's connection state to broadcastlink server.
