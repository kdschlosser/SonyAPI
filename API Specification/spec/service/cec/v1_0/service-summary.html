Service cec Â¶

This service handles APIs that are related to CEC functions.
    API List
    API name Auth Level Description
    setCecControlMode (v1.0)
    Generic
    Deprecated.

    This API must not be reused by servers except listed in supported devices. (This API is replaced by v1_0.setCecControlSettings.)
    setMhlAutoInputChangeMode (v1.0)
    Generic
    This API provides a function to set the mode of auto input change for MHL.
    setMhlPowerFeedMode (v1.0)
    Generic
    This API provides a function to change power feed mode for MHL.
    setPowerSyncMode (v1.0)
    Generic
    Deprecated.

    This API must not be reused by servers except listed in supported devices. (This API is replaced by v1_0.setCecControlSettings.)
    Common parameters in cec service
    Control Mode
    Set whether to use HDMI device control function.
    "on"
    Enable CEC control function.
    "off"
    Disable CEC control function.
    Audio Return Channel
    Sets Enable/Disable of Audio Return Channel function.
    "auto"
    Enable ARC function. Digital audio of TV is input via HDMI automatically.
    "off"
    Disable ARC function.
    Power-Off sync mode
    Power-Off sync mode
    "on"
    When this device is turned off, CEC devices connected to this device also gets turned off automatically.
    "off"
    This mode is disabled.
    Power-On sync mode
    Power-On sync mode.
    "on"
    When any CEC devices connected to this device are turned on, this device also automatically get turned on.
    "off"
    This mode is disabled.
