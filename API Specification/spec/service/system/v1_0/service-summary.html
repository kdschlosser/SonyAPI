Service system Â¶

This service handles APIs that are related to basic device functions like as follows.
  • To provide device related information like category, serial number and so on.
  • To provide/change general setting information like network setting, power status, clock setting and so on
  • To provide function to access general component to Sony device, like Remote Commander.

    Common parameters in system service
    Front Panel Brightness
    Set Brightness of the system display.
    "bright"
    Front panel brightness is bright.
    "dark"
    Front panel brightness is dark.
    "off"
    Front panel brightness is off.
    AV Separation
    Set whether to enable AV Separation Output Mode or not.
    "on"
    Output data in the AV Separation Output Mode.
    "off"
    Output data in the Multi Output Mode.
    Auto Display
    Auto Display settings.
    "on"
    OSD is displayed automatically when state or mode etc. is changed.
    "off"
    OSD is displayed only by user operation.
    Screen Saver Mode
    Set whether to use Screen Saver feature or not.
    "on"
    Enable Screen Saver.
    "off"
    Disable Screen Saver.
    Software Update Notification Mode
    Set whether to use Network update notification function or not.
    "on"
    Enable Network update notification function.
    "off"
    Disable Network update notification function.
    Rich Meta Information (Gracenote) settings
    Set for connecting to Gracenote.
    "auto"
    When playback is stopped, BD ROM/DVD ROM creates TOC for Gracenote connection and obtains information from Gracenote. Whereas CD DA creates TOC and obtains information from Gracenote upon mounting.
    "manual"
    Creates TOC and obtains information from Gracenote only when there is a request for information acquisition, such as window display.
    Auto Renderer
    Set the access permission setting value required when registering new DMC/+PU+, DMR and DMP.
    "on"
    When there is an access from new DMC/+PU+ (Send contents etc.), DMR and DMP registers Access Permission as "Allow" state. When it exceeds the max number of possible Access Permissions, is registers Access Permission as "Block(Hold)" state.
    "off"
    When there is an access from new DMC/+PU+ (Send contents etc.), DMR and DMP registers Access Permission as "Block(Hold)" state.
    Name of a device.
    Name of a device.
    • (ex) "BD Player in Living"
    IR Repeat
    Set whether a server receives and sends IR signal to other device.
    "on"
    A server receives and sends IR signal for other device.
    "off"
    A server does not receive and send IR signal to other device.
    CIS-IP Control
    Set whether a dedicated controller (home automation controller) on the home network is allowed to control a device by CIS-IP control.
    "on"
    Enable CIS-IP control function
    "off"
    Disable CIS-IP control function
    Auto Update
    Set whether to enable auto software update function or not.
    "on"
    Enable Auto SW Update function
    "off"
    Disable Auto SW Update function
    Time Zone settings
    Set Time Zone unique name (Area/Location) used in tz database or offset value is also able to be added by the following format.
    <Time Zone unique name used in tz database>|<optional offset value (minutes)>
    • (ex) "America/New_York|-240"
    • (ex) "Asia/Tokyo|540"
    OSD Language
    OSD language code represented by BCP47 (RFC4646 and RFC4647)
    • (ex) "ja"
    • (ex) "en-US"
    Client Language
    Client language code represented by BCP47 (RFC4646 and RFC4647)
    • (ex) "ja"
    • (ex) "en-US"
    Quick Start Mode
    Set whether to set Standby mode during Power Off or not.
    "on"
    Standby mode is set during Power Off.
    "off"
    Standby mode is not set during Power Off.
    Auto Standby
    A feature that this device automatically changes to Power OFF, when judged that there is no operation for a certain period of time. Set whether to Auto Standby feature or not.
    "on"
    Enable Auto Standby.
    "off"
    Disable Auto Standby.
    WOL mode
    Set whether to enable WOL mode settings.
    "on"
    Enable WOL mode.
    "off"
    Disable WOL mode.
    Network standby mode
    Set whether to enable Network standby mode. When a device is Network standby state, network is available even in power off state.
    "on"
    Enable Network standby mode.
    "off"
    Disable Network standby mode.
    Secure Link mode
    Set Secure Link to a specific surround amplifier in range of wireless reception.
    "on"
    Enable Secure Link function.
    "off"
    Disable Secure Link function.
