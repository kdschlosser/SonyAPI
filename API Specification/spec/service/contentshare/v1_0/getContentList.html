contentshare - v1.0 Private

getContentList Â¶

This API provides a function to get the list of contents uploaded to server.
Note
The process of creating thumbnail image from original content depends on implementation.
Syntax
http://<Base URL>/contentshare
ws://<Base URL>/contentshare
Authentication Level
private
  • Request
  • Response
  • Notification
  • See also
params' Elements
An object composed by following pair(s).
name type multiplicity default description
index integer ? 0
Start index to get list items.
count integer ? 50
Count of the maximum number of items that can be listed, starting from "index".
type string-array ? null
Optional parameter to narrow down result. Following values are defined.
  • "photo" - photo.
  • "video" - video
  • "music" - music
  • null - Not specified. (all types of contents)
uuid string ? null
Optional parameter to narrow down the result only belonging to specified uuid. In case null is set, all contents are returned.
JSON Example
{
    "id": 33,
    "method": "getContentList",
    "params": [{
        "index": 0,
        "count": 50,
        "uuid": "20ED712E-8C43-4FCF-9712-XXXXXXXXXXXA",
        "type": [
            "video",
            "photo"
        ]
    }],
    "version": "1.0"
}
result's Elements
An array of objects composed by following pairs.
name type multiplicity default description
index integer 1  
Index of the list, starting with "index" indicated in the request.
fileName string 1  
Filename of the content. (unique key used in playContent)
fileNameAlias string 1  
Alias of filename of the content. (unique key used in playContent)
uuid string 1  
Client uuid.
userNickname string 1  
Nickname of a client that can be configured by user to display
originalUrl string 1  
URL of the original content uploaded by client.
thumbnailUrl string 1  
URL of the content converted by server for thumbnail use.
originalOrientation integer 1  
Orientation of the original content uploaded by client. Value definition is same as exif orientation attribute.
thumbnailOrientation integer 1  
Orientation of the content converted by server for thumbnail use.Value definition is same as exif orientation attribute.
artistName string 1  
Artist name of the content.
status string 1  
playing status of the content.
  • "stopped" - stopped
  • "playing" - playing.
  • "paused" - paused
type string 1  
content type
  • "photo" - photo.
  • "video" - video
  • "music" - music
votedListByUuid string-array 1  
List of uuid who has voted on this content.
size integer 1  
Content size (byte).

code Explanation

ContentShare application is not running.
41801
Count is out of range.
41802
Index is out of range.
