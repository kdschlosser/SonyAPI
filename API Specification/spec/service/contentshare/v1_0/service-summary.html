Service contentshare Â¶

This service handles APIs that are related to ContentShare application's functions. ContentShare provides the service to share various types of contents to multiple clients via server device or to server device itself. Note that general HTTP request is used for client to post content to server and to download image from server.
    API List
    API name Auth Level Description
    closeContentShare (v1.0)
    Generic
    This API provides a function to terminate ContentShare application.
    getContentList (v1.0)
    Private
    This API provides a function to get the list of contents uploaded to server.
    getContentShareServerInfo (v1.0)
    Private
    This API provides a function to get information to access to ContentShare application.
    getUserList (v1.0)
    Private
    This API provides a function to get user information in ContentShare application.
    requestToNotifyEvent (v1.0)
    Private
    This API provides a function to request device to start monitoring of events which will be notified from ContentShare application.
    rotatePhoto (v1.0)
    Generic
    This API provides a function to rotate photo in ContentShare application.
    setBgmControlMode (v1.0)
    Generic
    This API provides a function to play BGM or not when ContentShare application is running.
    setQuickInvitationMode (v1.0)
    Generic
    This API provides a function to request ContentShare application to show/hide quick invitation information which is necessary for client to connect with ContentShare server.
    setUserNickName (v1.0)
    Generic
    This API provides a function to set/update client's nickname related to uuid.
    togglePlayStatus (v1.0)
    Generic
    This API provides a function to toggle play status of content located in a server device where ContentShare application is running.
    voteContent (v1.0)
    Generic
    This API provides a function to vote on content.
