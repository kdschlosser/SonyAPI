contentshare - v1.0 Generic

togglePlayStatus Â¶

This API provides a function to toggle play status of content located in a server device where ContentShare application is running.
Syntax
http://<Base URL>/contentshare
ws://<Base URL>/contentshare
Authentication Level
generic
  • Request
  • Response
  • Notification
  • See also
params' Elements
An object composed by following pair(s).
name type multiplicity default description
fileName string 1  
The file name of the content to show. Client can get the list of file name by GetContentList API. The files to be served are managed by ContentShare service and unique file name is assigned.
JSON Example
{
    "id": 52,
    "method": "togglePlayStatus",
    "params": [{"fileName": "xxxxx"}],
    "version": "1.0"
}
result's Elements
Not necessary.
JSON Example
{
    "id": 52,
    "result": []
}
Error Code
Following error codes are important for this API. Refer to error code for other errors.
code Explanation
7
ContentShare application is not running.
41804
File not found.
