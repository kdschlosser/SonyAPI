Service photoshare Â¶

This service handles APIs that are related to PhotoShare application's functions. PhotoShare provides the service to share image type contents to multiple clients via server device or to server device itself. Note that general HTTP request is used for client to post image to server and to download image from server.
    API List
    API name Auth Level Description
    closePhotoShare (v1.0)
    Generic
    This API provides a function to terminate PhotoShare application.
    getPhotoList (v1.0)
    Private
    This API provides a function to get the list of photos uploaded to server.
    getPhotoShareServerInfo (v1.0)
    Private
    This API provides a function to get information to access to PhotoShare application.
    playPhotoContent (v1.0)
    Generic
    This API provides a function to show a photo located in a server device where PhotoShare application is running.
    requestToNotifyEvent (v1.0)
    Private
    This API provides a function to request device to start monitoring of events which will be notified from PhotoShare application.
    setBgmControlMode (v1.0)
    Generic
    This API provides a function to play BGM or not when PhotoShare application is running.
    setQuickInvitationMode (v1.0)
    Generic
    This API provides a function to request PhotoShare application to show/hide quick invitation information which is necessary for client to connect with PhotoShare server.
    setUserNickName (v1.0)
    Generic
    This API provides a function to set/update client's nickname related to uuid.
