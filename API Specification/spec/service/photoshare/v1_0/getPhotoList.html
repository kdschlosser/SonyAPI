photoshare - v1.0 Private

getPhotoList Â¶

This API provides a function to get the list of photos uploaded to server.
Note
The process of creating thumbnail image from original image depends on implementation.
Syntax
http://<Base URL>/photoshare
ws://<Base URL>/photoshare
Authentication Level
private
  • Request
  • Response
  • Notification
  • See also
params' Elements
An object composed by following pair(s).
name type multiplicity default description
index integer ? 0
Start index to get list items.
count integer ? 50
Count of the maximum number of items that can be listed, starting from "index".
JSON Example
{
    "id": 50,
    "method": "getPhotoList",
    "params": [{
        "index": 0,
        "count": 55
    }],
    "version": "1.0"
}
result's Elements
An array of objects composed by following pairs.
name type multiplicity default description
index integer 1  
Index of the list, starting with "index" indicated in the request.
fileName string 1  
Filename of the photo. (unique key used in playPhotoContent)
uuid string 1  
Client uuid.
userNickname string 1  
Nickname of a client that can be configured by user to display
originalUrl string 1  
URL of the original photo uploaded by client.
thumbnailUrl string 1  
URL of the photo converted by server for thumbnail use.
originalOrientation integer 1  
Orientation of the original photo uploaded by client. Value definition is same as exif orientation attribute.
thumbnailOrientation integer 1  
Orientation of the photo converted by server for thumbnail use.Value definition is same as exif orientation attribute.

code Explanation

PhotoShare application is not running.
41801
Count is out of range.
41802
